Lunghezza dell'Output

La LUNGHEZZA DELL'OUTPUT delle funzioni HASH dipende dall'algoritmo utilizzato e fornisce un EQUILIBRIO TRA SICUREZZA ED EFFICIENZA.

In generale, le funzioni HASH producono un output di lunghezza fissa, che può variare da algoritmo ad algoritmo.

PIÙ È LUNGA LA STRINGA DIGITALE, MAGGIORE È LA SICUREZZA, MINORE È LA PROBABILITÀ DI COLLISIONI e sarà, dunque, difficile invertire l'HASH.

Le lunghezze degli output degli algoritmi HASH più utilizzati sono:

  • MD5 (Message Digest Algorithm 5): Produce un HASH di 128 bit, equivalente a 16 byte o 32 caratteri esadecimali.

  • SHA-1 (Secure HASH Algorithm 1): Produce un HASH di 160 bit, equivalente a 20 byte o 40 caratteri esadecimali.

  • SHA-256 (Secure HASH Algorithm 256 bit): Produce un HASH di 256 bit, equivalente a 32 byte o 64 caratteri esadecimali.

  • SHA-512 (Secure HASH Algorithm 512 bit): Produce un HASH di 512 bit, equivalente a 64 byte o 128 caratteri esadecimali.
